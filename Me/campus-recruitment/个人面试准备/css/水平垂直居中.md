# CSS的垂直居中和水平居中
  > 垂直与水平居中，一般在面试过程中作为基础的基础，所以说垂直与水平居中肯定是需要前端开发者熟练于心得基础

## 水平居中

### 行内元素-水平居中&垂直居中
  > span、a、lable、strong、b

#### 行内-水平居中

  1. text-align: center

#### 行内-垂直居中

  1. vertical-align: middle
    > 把此元素放置在父元素的中部,只有一个元素属于inline或是inline-block
  
  2. line-height
    > 设置行高与 元素等高，即可达到 行内元素居中

### 块级元素-水平居中&垂直居中

  > div、p、li、h1、h2、h3、h4等html标签都是块级元素

#### 块级-水平居中

  1. flex
    > 弹性布局
```js
  <style>
    #main {
        width: 400px;
        height: 150px;
        border: 1px solid #c3c3c3;
        display: flex;
        justify-content: center;// 项目位于容器的中心,达到水平居中效果
    }
    #main div {
        width: 70px;
        height: 70px;
    }
    <div id="main">
      <div style="background-color:coral;"></div>
      <div style="background-color:lightblue;"></div>
      <div style="background-color:khaki;"></div>
      <div style="background-color:pink;"></div>
    </div>
  </style>
```
  2. 绝对定位 + transform: translate()
```js
<style>
  .out{
    width: 500px;
    height: 500px;
    background: red;
    position: relative;
    margin: auto;
  }
  .inner{
    width: 200px;
    height: 200px;
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
    background: blue;
  }
</style>
<body>
  <div class="out">
    <div class="inner"></div>
  </div>
</body>
```
  3. 绝对定位 + margin: auto + left: 0 , right: 0

```js
<style>
  .out{
    width: 500px;
    height: 500px;
    background: red;
    position: relative;
    margin: auto;
  }
  .inner{
    width: 200px;
    height: 200px;
    position: absolute;
    left: 0;
    right: 0;
    margin: auto;
    background: blue;
  }
</style>
<body>
  <div class="out">
    <div class="inner"></div>
  </div>
</body>
```
  4.  display: gird + 子元素 align-self: center;

```js
<style>
  .out{
    width: 500px;
    height: 500px;
    background: red;
    display: grid;
  }
  .inner{
    width: 200px;
    height: 200px;
    align-self: center;
    background: blue;
  }
</style>
<body>
  <div class="out">
    <div class="inner"></div>
  </div>
</body>
```

#### 块级-垂直居中

1. flex
    > 弹性布局
```js
  <style>
    #main {
        width: 400px;
        height: 150px;
        border: 1px solid #c3c3c3;
        display: flex;
        align-items: center;// 项目位于容器的中心,达到水平居中效果
    }
    #main div {
        width: 70px;
        height: 70px;
    }
    <div id="main">
      <div style="background-color:coral;"></div>
      <div style="background-color:lightblue;"></div>
      <div style="background-color:khaki;"></div>
      <div style="background-color:pink;"></div>
    </div>
  </style>
```
  2. 绝对定位 + transform: translate()
```js
<style>
  .out{
    width: 500px;
    height: 500px;
    background: red;
    position: relative;
    margin: auto;
  }
  .inner{

    height: 200px;
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
    background: blue;
  }
</style>
<body>
  <div class="out">

  </div>

```
  3. 绝对定位 + margin: auto + top: 0 , bottom: 0
